# Software renderer
A toy software implementation of a graphics pipeline with programmable vertex and fragment shaders.

## Prerequisites
 - CPU with AVX
 - OpenGL 4.5, GLEW, GLFW
 - clang / GCC compiler with C++14 support
 - Linux
